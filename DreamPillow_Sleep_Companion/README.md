# DreamPillow - Sleep & Relaxation Companion

## Objective

This project is a simple college laboratory exercise that demonstrates a modern wellness-themed webpage using semantic HTML5 elements, HTML5 audio and video tags, and responsive CSS3 styling.

## Technologies Used

- HTML5
- CSS3

## Folder Structure

```text
DreamPillow_Sleep_Companion/
|-- index.html
|-- style.css
|-- README.md
`-- assets/
    |-- images/
    |   |-- logo.png
    |   `-- hero.jpg
    |-- audio/
    |   `-- relaxing-music.mp3
    `-- video/
        `-- sleep-guide.mp4
```

## Instructions to Run

1. Open the `DreamPillow_Sleep_Companion` folder.
2. Double-click `index.html` or open it in any web browser.
3. If real media files are added later, keep the same filenames and folder paths.
