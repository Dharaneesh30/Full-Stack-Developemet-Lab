# DreamPillow Random Product Viewer

## Objective

This project is a beginner-friendly React laboratory exercise that randomly selects one DreamPillow product from an array of 10 JSON objects and displays all of its properties dynamically in a table.

## Technologies Used

- React
- JavaScript
- JSX
- CSS
- React Hooks (`useState`)
- Props

## Folder Structure

```text
RandomObjectReact/
├── public/
│   └── index.html
├── src/
│   ├── App.js
│   ├── App.css
│   ├── DisplayObject.js
│   ├── RandomObject.js
│   └── index.js
└── package.json
```

## Steps to Run

1. Open the terminal inside the `RandomObjectReact` folder.
2. Install the dependencies:

```bash
npm install
```

3. Start the React development server:

```bash
npm start
```

4. Open the browser and view the app at:

```text
http://localhost:3000
```
