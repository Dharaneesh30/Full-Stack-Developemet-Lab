# DreamPillow Product Showcase

This project is a simple responsive Bootstrap 5 webpage created for a college laboratory exercise.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (CDN)

## Bootstrap Components Used

- Responsive Navbar
- Grid System
- Cards
- Table
- Buttons

## Folder Structure

```text
Bootstrap_DreamPillow/
|-- index.html
|-- style.css
|-- README.md
`-- assets/
    `-- images/
        `-- pillow.jpg
```

## How to Run

1. Open the `Bootstrap_DreamPillow` folder.
2. Double-click `index.html` or open it in any web browser.
3. Make sure the internet connection is available so the Bootstrap CDN loads correctly.
